    main线程以及新建线程的默认线程组是main线程组
    线程组的优先级默认为10
    修改父线程组的最大优先级,子线程组也随之修改

        当然超过优先级访问依然抛异常
    创建线程的时候,它的优先级<=它的线程组最大优先级
